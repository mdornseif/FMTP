
BUILD EXE
---------

install python 2.5, python 2.6 or python 2.7
install pywin32 latest
install wxpython 2.8.latest
install simplejson, if you are not using Python 2.7
install py2exe latest
install ghostscript and gsview
run python setup.py


BUILD INSTALLER
---------------

install nullsoft install system 2.x (i've used 2.46)
edit printclient.nsi and make sure it points to your ghostscript and gsview folders
right click printclient.nsi -> compile


thats it
